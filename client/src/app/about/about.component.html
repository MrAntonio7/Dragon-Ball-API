What is this?

This website provides a RESTful API interface to highly detailed objects built from thousands of lines of data related to Dragon Ball. We specifically cover the anime franchise, though we'd like to cover the comic. Using this website, you can consume information on Dragon Ball, characters, transformations, fusions and much more.

This API is "public", because we need a APIKEY but we can obtein the APIKEY for FREE simply by registering. Once registered and logged in your profile, you will get your APIKEY, and completely free!.

What exactly is an API?

This is probably worth clearing up very quickly for those who don't know. An API (Application Programming Interface) is a set of interfaces (in this case, url links) that are publicly available, that allow developers to interact with an application. In this instance; the application is a database of thousands of FragonBall-related objects.

What are the features?

I'm using Phalcon, a PHP Framework , to serve a REST API to you. The interface is made with Angular 5.

The data is all formatted in JSON and we also support JSON Schema for programmatically understanding the attributes of each resource.

Why did you build this?

I realized that Phalcon is a very fast and clean code framework. I looked for APIs about this franchise and I did not find any! I also had an end-of-course project, so I took advantage of the moment.
