Documentation

Introduction

Welcome to the DragonBallAPI! The unofficial API of Dragon Ball. This documentation should help you familiarise yourself with the resources available and how to consume them with HTTP requests. Read through the getting started section before you dive in. Most of your problems should be solved just by reading through it.

Getting Started

Let's make our first API request to the Star Wars API!.

This API is "public", because we need a APIKEY but we can obtein the APIKEY for FREE simply by registering. Once registered and logged in your profile, you will get your APIKEY, and completely free!.

Let's make our first API request to the Star Wars API!

Open up a terminal and use curl or use in your code. Don't forget to put the apikey parameter.

  curl -i -H "Accept: application/json" -H "Content-Type: application/json" -X GET  http://dragonballapi.com/api/characters?apikey=[YOUR_APIKEY]

Base URL

The Base URL is the root URL for all of the API, if you ever make a request and you get back a 404 NOT FOUND response then check the Base URL first.

The Base URL for Dragon Ball API is:

  http://dragonballapi.com/api/

Authentication

Dragon Ball API is an "open" API, only need the APIKEY and is free!. No authentication is required to query and get data, only APIKEY. If you find a mistake in the data, send email to the author.

JSON Schema

All resources support JSON Schema. Making a request to /api/[resource] will give you the details of that resource. This will allow you to programmatically inspect the attributes of that resource and their types.

Resources

Characters

A character recurse is a individual person or character within the Dragon Ball universe.

Links:
  • /characters -- get all the character resources
  • /characters/[id] -- get a specific character resource
Example:
  http://dragonballapi.com/api/characters/3?apikey=[YOUR_APIKEY]
Attributes:
id_character :number ID of character resource
name :string Name of character resource
id_race :number ID of race of character resource
race :string Name of race of character resource
gener :string Gener of character resource
img :string Image's link of character resource
weight :string Weight of character resource
height :string Height of character resource
birth :string Birth's Date of character resource
dead :string Dead's Dates of character resource
resurrection :string Resurrecition's Dates of character resource
job :string Jobs of character resource
alliances :string Alliances of character resource
description :string Description of character resource
fusion :string Fusion of character resource
transformation :string Transformations of character resource
apparition :string Apparitions of character resource
planet :string Name of planet of character resource
